Connectivity & Privacy

🔵 BLE
Bluetooth Low Energy 100% local. Commands go directly to the device over Bluetooth. Nothing ever leaves your machine.
🟢 LAN
Local Wi-Fi HTTP (:9000) 100% local. Talks directly to the device's built-in HTTP server. No internet, no account. WiFi-capable devices only.
🟡 Cloud
Divoom Cloud (appin.divoom-gz.com) Sends commands to Divoom's servers. Required for: gallery browsing, clock face store, community features. Requires a Divoom account.
🔴 External
External Internet APIs Used for weather, stock prices, album art lookups. 3rd-party services — no Divoom account required. Data is anonymous.
